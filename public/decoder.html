Ciphermania - Decoder

Decoding is the process of interpreting the encrypted message using an understanding/key of the cipher used. This essential step is done by the recipient, who can now privately translate the message for their understanding, finishing the process of privately communicating a message.

Use the decoder below by first inputting the message you want to decrypt in the first input field. Then, based on the type of cipher, input any correct key that is required to encode your message. For example, if you are using the Caesar Cipher, type the Caesar shift (defined in the library) into the second input field. Finally, select the type of cipher to decode.
