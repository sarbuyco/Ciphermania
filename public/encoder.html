Ciphermania - Encoder

Encoding is the process of encrypting your message and translating it according to the form and structure of a cipher. This special process is the essential part of securing your message before it is interpreted by the recipient, ensuring the content is private and not easily understood by unwanted eyes.

Use the encoder below by first inputting the message you want to encrypt in the first input field. Then, based on the type of cipher, input any correct key that is required to encode your message. For example, if you are using the Caesar Cipher, type the Caesar shift (defined in the library) into the second input field. Then select the type of cipher to encode.



Notes:
1. Add a key for Caesar and Affine ciphers.
2. Apphine needs two inputs, a slope (must be coprime to 26) and an intercept. Seperate through space (i.e. "17 20")
3. (For decoding) Bacon only has lowercase a and b as keys for this site.
